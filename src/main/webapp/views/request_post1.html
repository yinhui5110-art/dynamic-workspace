요청을 보낼 수 있는 페이지

		특징:
		1. POST방식으로 요청을 보내면 BODY영역에 데이터를 포함시켜 요청
		=> 사용자가 입력한 값(데이터)들이 URL에 노출되지 않는다
		=> 로그인, 회원가입 같은 기능들은 POST방식이 적합하다
		
		2. 전송하는 길이에 제약이 없음
		=> 즉, 게시글 작성 같은 경우 POST방식이 적합하다
		
		3. 최대 요청 받는 시간이 존재하다
		
		4. 새롭게 데이터를 만들어내는 기능에 사용된다
